# export-nideriji

导出 [你的日记](http://nideriji.com/) APP 日记数据

> 应用抓包发现数据实际来源于 [Oh 生活](https://ohshenghuo.com/)

## Format

- 通用 JSON 格式
- [一本日记](http://1diary.me)
